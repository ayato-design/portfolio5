# LP型企業サイト模写(WordPressテーマ化)

## 概要
HTMLで制作したコーポレートサイトを元に、WordPressのオリジナルテーマとして再構築しました。
静的なコーディングに加え、投稿機能(ニュース)の実装により、更新性を意識したサイトとして構築しています。
お問い合わせフォームにはContact Form 7を使用し、運用を想定した構成にしています。

## デモ
HTML版：
https://ayato-design.github.io/portfolio3/

※WordPress版はローカル環境で構築しています。

## 制作目的
- WordPress開発/運用に慣れる為

## 担当範囲
- トップページのコーディングおよびテンプレート化
- 投稿機能を用いたニュース一覧(home.php)・詳細ページ(single.php)の実装
- header.php / footer.phpの分割による共通パーツ化
- WordPressループを用いた記事出力

## 工夫した点
- お借りした素材は使わず、全て自分で用意しました
- 作成するデザインを改めて考え、デザインカンプは作り直しました
- 配布ページのヒントを見ながら、自分なりに答えを出し作成しました
- コンタクトページ、ニュースページは自作デザインです(追加)

## 苦労した点・解決方法
- 久し振りにPHPとWordPressに触れる為、わからない事だらけで制作に時間が掛かりました
- わからない部分は主にネットで検索して解決しました
- それでもわからない場合は、AI(Chat GPT/Copilot/Claud)を使用して解決しました

## 今後改善したい点
- カスタム投稿タイプやカスタムフィールドの実装までは対応出来ていない為、今後の課題としています
- バリデーションやセキュリティ面の理解をさらに深めていきたいと考えています
